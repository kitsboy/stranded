# Internationalization — Stranded

**Status (2026-07-15 / v2.3.7):** Client locale toggle for **nav, home hero, footer, sites filters, and mission panel**.

## Supported UI locales

| Code | Label | Coverage |
|------|--------|----------|
| `en` | English | Default · full product language |
| `fr` | Français | Nav, hero, footer, sites, mission |
| `de` | Deutsch | Nav, hero, footer, sites, mission |
| `es` | Español | Nav, hero, footer, sites, mission |

Source: `lib/i18n.ts` · Hook: `lib/useLocale.ts` · Control: `components/LanguageToggle.tsx`  
Consumers: `components/Nav.tsx`, `components/Footer.tsx`, `app/page.tsx`, `app/sites/page.tsx`, `components/MissionPanel.tsx`

## Behaviour

- Default: **English** (`en`)  
- Stored in `localStorage` key `stranded-locale`  
- `stranded-locale-change` custom event syncs all `useLocale()` consumers  
- Menu opens **below** the control (never above sticky nav)  
- Click to open; Escape / outside click to close  
- Invalid stored values reset to `en`

## Not yet translated

- Full home page body (problem/opportunity sections, flow, personas)  
- Map chrome, education content, pitch deck copy  
- Marketing Hub HTML  
- SEO locale files under `docs/SEO-*.md` are marketing notes, not wired to the app toggle  

## Relation to Give A Bit master i18n

Give A Bit may host a larger translation system under `~/projects/giveabit`. Stranded currently keeps a **minimal local** string table for speed and static export. Do not assume Give A Bit master files drive this site until explicitly wired.
