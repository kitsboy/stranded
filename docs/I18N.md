# Internationalization — Stranded

**Status (2026-07-09 / v2.3.5):** Client locale toggle for **nav labels only**.

## Supported UI locales

| Code | Label | Coverage |
|------|--------|----------|
| `en` | English | Default · full product language |
| `fr` | Français | Nav strings |
| `de` | Deutsch | Nav strings |
| `es` | Español | Nav strings |

Source: `lib/i18n.ts` · Control: `components/LanguageToggle.tsx` · Consumer: `components/Nav.tsx`

## Behaviour

- Default: **English** (`en`)  
- Stored in `localStorage` key `stranded-locale`  
- Menu opens **below** the control (never above sticky nav)  
- Click to open; Escape / outside click to close  
- Invalid stored values reset to `en`

## Not yet translated

- Page bodies (home, map chrome, education, pitch, methodology copy)  
- Marketing Hub HTML  
- SEO locale files under `docs/SEO-*.md` are marketing notes, not wired to the app toggle  

## Relation to Give A Bit master i18n

Give A Bit may host a larger translation system under `~/projects/giveabit`. Stranded currently keeps a **minimal local** string table for speed and static export. Do not assume Give A Bit master files drive this site until explicitly wired.
